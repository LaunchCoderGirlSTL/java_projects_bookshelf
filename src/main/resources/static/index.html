A working demo of your technology learning goals

This "bookshelf" application is also a deployed model of a working app with Spring MVC, SQL, and front-end pages as well as the host of exercises for many of the lessons
  • Hosted on the web with Heroku
  • A web application run by spring-boot
  • A collection of web servlets configured using Spring MVC
  • Uses an in-memory database (H2) to store the results of data added to the application
  • Uses SQL expressions, JDBC Template, and Spring AOP to issue database queries from Java code
